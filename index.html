Disney Junior Birthday Book

The Birthday Book lets guests upload their kid's pictures to our online gallery. Some lucky children even get to appear on-air! This was a very unique in-house product which I developed end-to-end and continue to product manage.

Roles:

  • Product Owner
  • Front-end Development
  • Back-end Development
  • Design
  • UX Design
  • IA Design
  • Upload API management
  • Documentation

Next: Birthday Book IA

Disney Junior Birthday Book: IA

The fairly complex user flow required mapping out in order to properly illustrate how the guest interacts with various APIs

Asynchronous calls to the user database helps move the guest quicker through the process and email verification keeps them informed throughout the process.

Key Points:

  • Asynchronous calls
  • Email communication for onward journeys
  • Simple Task focused layout


Next: Birthday Book UX

Disney Junior Birthday Book: UX

By implementing an Agile iterative design workflow and using guest feedback from live testing, various UX improvements have been successfully integrated post-launch.

Key UX improvements:

  • A number of entries were from under 18s as parents thought they needed to register their child. A new modal and age check at login was introduced reducing moderator resource and complaints.
  • A more robust realtime error validation upload form.
  • Seamless mobile upload experience
  • CSS 3 animations for that extra Disney magic.
  • Fully responsive HTML framework.

Next: Birthday Book Mobile First

Disney Junior Birthday Book: Mobile First

HTML mobile uploading and access to the internal camera and file system was finally made possible with the introduction of iOS6. Little online support for mobile file uploading at the time meant hours of prototype development and testing.

The majority of guests would be using smart phones to take pictures so the best solution was a mobile first HTML file uploader with flash fallback.


Next: Birthday Book Results

Disney Junior Birthday Book: Results

Mobile access to the site was 20% higher than the overall Disney.co.uk average, with 65% of uploads coming from mobile.

Our core audience were mothers with smartphones and the results clearly show with over 80% of uploaders being female.

Since launch the Birthday Book has received over 30 000 approved uploads with an average of 1500 per month.

During the first few months of launch there were a number of recurring issues from guests. Small code, workflow and communication tweaks have cutdown complaints and issues by 90%.
